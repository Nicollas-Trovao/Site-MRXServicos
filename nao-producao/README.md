# Nao producao

Esta pasta guarda apenas arquivos de teste, prototipos e rascunhos que nao devem entrar na raiz publica do site.

- `prototipo.html`: pagina para experimentar componentes e ideias.
- `css/prototipo.css`: estilos usados apenas pelo prototipo.

As paginas finais ficam na raiz do projeto. Quando algo daqui for aprovado, copie apenas o necessario para a estrutura final.
